# Random Quote Generator

A simple web application that generates a random quote when the user clicks a button. The project demonstrates how a frontend interface communicates with a backend API built using Python and Flask.

---

## Overview

This project is a beginner-friendly full-stack application where the frontend sends a request to a backend server, and the server responds with a randomly selected quote. The quote is then displayed on the webpage dynamically.

---

## Features

* Generate a random quote with a single button click
* Backend API built using Python and Flask
* Frontend built with HTML, CSS, and JavaScript
* Demonstrates basic API communication between frontend and backend

---

## How It Works

1. The user clicks the **Generate Quote** button on the webpage.
2. The frontend JavaScript sends a request to the backend API.
3. The backend server processes the request and selects a random quote.
4. The quote is returned to the frontend and displayed on the webpage.

---

## Running the Project Locally

1. Install Python on your system.
2. Install Flask if not already installed.
3. Run the backend server:

```
python server.py
```

4. Open `index.html` in your browser.
5. Click the **Generate Quote** button to see a random quote.
